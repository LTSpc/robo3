 ####  git関連追加 ####
-過去の状態に戻る
まず過去に戻りたいブランチを選択する(現在のブランチの過去であれば不要）
git checkout [ブランチ]
git logで現在のブランチの過去の状態を確認して，
戻りたい状態のハッシュをコピーして以下の通り実行
git checkout [ハッシュ]

-この状態を別のブランチとして編集していく
git add .
git commit -m ‘start lesson10’
git push —set-upstream origin lesson10

#### git コマンド ####
設定の確認
cat ~/.gitconfig

github上のものをclone ※sshを使用すること
git clone git@github.com:LTSpc/robo2025.git

gitに追加する準備
git add test.txt

addできたか確認
git status

コミットする
git commit -m 'Add a command'

コミットのログを確認
git log -n 1

リモートリポジトリにプッシュ
git push

~/.ssh/config ->
Host github.com
        Hostname ssh.github.com
        User git
        IdentityFile ~/.ssh/id_rsa
        Port 443
        ProxyCommand nc -X connect -x 192.168.58.34:80 %h %p


robo2025/.git/config ->
[core]
        repositoryformatversion = 0
        filemode = true
        bare = false
        logallrefupdates = true
[remote "origin"]
        url = git@github.com:LTSpc/robo2025.git
#       url = https://github.com/LTSpc/robo2025.git
        fetch = +refs/heads/*:refs/remotes/origin/*
[branch "main"]
        remote = origin
        merge = refs/heads/main
